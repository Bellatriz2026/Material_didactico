Explicación paso a paso

Abre cada tema para ver cuándo se usa y un ejemplo.

BUSCARV

BUSCARV busca un dato en la primera columna de una tabla y devuelve información que está hacia la derecha. Sirve cuando tienes un código y necesitas encontrar el producto, precio o categoría.

=BUSCARV(A2;A6:F10;4;FALSO)

Lectura fácil: busca el código de A2 en la tabla A6:F10 y devuelve el dato de la columna 4. FALSO significa búsqueda exacta.

A2 valor buscado A6:F10 tabla 4 columna del precio FALSO coincidencia exacta

BUSCARX

BUSCARX es más flexible que BUSCARV. Puede buscar en una columna y devolver datos de otra columna, aunque no estén necesariamente hacia la derecha.

=BUSCARX(A2;A6:A10;D6:D10;"No encontrado")

Lectura fácil: busca el código de A2 en A6:A10 y devuelve el precio ubicado en D6:D10. Si no lo encuentra, escribe “No encontrado”.

CONCATENAR

CONCATENAR une textos de diferentes celdas. Es útil para crear nombres completos, mensajes o códigos personalizados.

=CONCATENAR(B2;" - ";D2)

Ejemplo: si B2 tiene “Audífonos” y D2 tiene “$12.00”, el resultado será: Audífonos - $12.00.

TEXTO

TEXTO convierte un valor en texto con un formato específico. Sirve para mostrar fechas, monedas o porcentajes de forma clara.

=TEXTO(D2;"$0.00")

Ejemplo: si D2 tiene 12, Excel puede mostrarlo como $12.00.

HIPERVÍNCULO

HIPERVÍNCULO crea un enlace dentro de Excel. Puede llevarte a una página web, un archivo, una hoja del mismo libro o un recurso de clase.

=HIPERVINCULO("https://www.office.com";"Abrir recurso")

Lectura fácil: abre el enlace indicado y muestra el texto “Abrir recurso”.
